# Java Text Editor without using IntelliJ GUI Designer

A example of using JetBrains IntelliJ to build a simple text editor from Swing components _**without using the UI Designer**_.

## Tools Used

| Tool     | Version  |
|----------|----------|
| Java     | 23.0.1.0 |
| IntelliJ | 2024.3.1 |
| VSCode   | 1.96.2   |
